Grid-template-areas

Określa obszary w układzie siatki. Jej elementy można nazwać korzystając
z grid-area, a następnie odwoływać się do ich nazwy za pomocą właściwości grid-template-areas.

Każdy obszar określony jest za pomocą apostrofów. A aby odnieść się do elementu bez nazwy należy użyć kropki.

Header
1
2
3
4
Aside
.grid-container {
grid-template-areas:
'header header header'
'. . aside'
'. . aside';
}

.header {
grid-area: header;
}

.aside {
grid-area: aside;
}
